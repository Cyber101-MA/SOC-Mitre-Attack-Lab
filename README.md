# SOC MITRE ATT&CK Lab

This repository documents a hands-on SOC lab focused on detecting, analyzing, and mapping real-world security events to MITRE ATT&CK techniques.

The lab simulates attacker behavior on Windows systems and validates detections using centralized logging and SIEM analysis.

## Objectives
- Understand how common attack techniques appear in logs  
- Detect and analyze malicious activity from a SOC perspective  
- Map alerts and behaviors to MITRE ATT&CK techniques  
- Build documented, repeatable detection workflows  

## Lab Environment
- Windows Server (Domain Controller)
- 2 Windows 10 Clients
- Centralized Log Management (SIEM) Ubuntu Linux
- Security Breaching Agent (Kali Linux)
